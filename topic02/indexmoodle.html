#Version Control

Never lose your work - even if the dog eats it!. Distributed version control allows you to securely save your work, share and colaborate with others. Several technologies exist that perform this task. We will look at one of the most popular - Git.
git-lab-01

The objective of this lab is to learn how to use Git to manage you project files and colabolrate with others.
github-pages-lab-01

The object of this short lab is to create a web site on Github to complement you project work.
markdown-lab-01

The objective of this lab is to learn Markdown, an easy-to-read, easy-to-write plain text format.
